## ⚙️ Changing the Data  

If you want to experiment with your own data, you only need basic file handling skills. Most examples use simple file formats like CSV or JSON. Here is a general way to switch data:  

1. Locate the sample data file in the example folder.  
2. Open the file in a text editor (e.g., Notepad or TextEdit) or spreadsheet app (e.g., Excel).  
3. Replace the sample values with your own data. Keep the same format and structure.  
4. Save the file.  
5. Reload the HTML file in your web browser to see your data displayed in the visuals.  

You do not need to write any code to do this.
